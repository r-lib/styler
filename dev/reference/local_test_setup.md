# Establish testing setup for current environment

Establish testing setup for current environment

## Usage

``` r
local_test_setup(cache = FALSE, .local_envir = parent.frame())
```

## Arguments

- cache:

  Whether or not to create and activate a cache in a temporary
  directory.

- .local_envir:

  The environment to use for scoping.
