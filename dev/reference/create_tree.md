# Create a tree from text

Create a tree representation from a text.

## Usage

``` r
create_tree(text, structure_only = FALSE)
```

## Arguments

- text:

  A character vector.

- structure_only:

  Whether or not create a tree that represents the structure of the
  expression without any information on the tokens. Useful to check
  whether two structures are identical.

## Value

A data frame.
