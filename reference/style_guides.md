# Style guides

Format code according to a style guide. Style guides are the input to
the argument `style` in
[`style_file()`](https://styler.r-lib.org/reference/style_file.md) and
friends. The available style guides are:

- the tidyverse style guide (see
  [`tidyverse_style()`](https://styler.r-lib.org/reference/tidyverse_style.md)).
