# Test `transformers_drop` for consistency

Check if the argument `transformers_drop` in
[`create_style_guide()`](https://styler.r-lib.org/reference/create_style_guide.md)
is consistent with the transformers specified in that function.

## Usage

``` r
test_transformers_drop(transformers)
```

## Arguments

- transformers:

  The output of
  [`create_style_guide()`](https://styler.r-lib.org/reference/create_style_guide.md)
  we want to test.
