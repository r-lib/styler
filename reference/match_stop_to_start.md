# Match a stop candidate to a start

Match a stop candidate to a start

## Usage

``` r
match_stop_to_start(start, stop_candidates)
```

## Arguments

- start:

  An integer.

- stop_candidates:

  Potential stop candidates.

## Examples

``` r
styler:::match_stop_to_start(1, c(3, 4, 5))
#> [1] 2
```
